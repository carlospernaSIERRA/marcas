La estructura de la página

Las etiquetas y su uso:

Las etiqutas suelen venir en pares: <p> y </p>, correspondiendo a <p> como la etiqueta de inicio y a </p> como la etiqueta final.
También, según la etiqueta se puede escribir de la siguiente manera <img/>

Las etiqutas que se usan en un documento HTML son las siguientes:

<!DOCTYPE html> Define el elemento como HTML5.
<HTML> Es el elemento raiz de la página HTML.
<head> Es el elemento que contiene la metainformación sobre el documento.
<title> Es el elemento que especifica un título para el documento.
<body> Es el elemento que contiene en contenido visible de la página.
<h1> Es el elemento que define un encabezado/titulo grande, dependiendo del numero que pongamos en un rango del 1 al 6, siendo 1 el más grande ira variando su tamaño.
<p> Es el elemento que define un párrafo.

El texto

<div> Permite justificar el texto de parrafo.
<b> Pone el texto en negrita.
<i> Pone el texto en cursiva.
<u> Subraya el texto.
<big> Incrementa el tamaño del texto.
<sup> Letra superíndice.
<sub> Letra subíndice.
<hr> Inserta una barra horizontal.
<br> Salto de línea.

Enlaces

La etiqueta <a> define un hipervínculo, que se utiliza para vincular de una página a otra.
El atributo más importante del elemento <a> es el atributo href, que indica el destino del enlace.

Por defecto, los enlaces aparecerán de la siguiente manera en todos los navegadores:

Un enlace no visitado está subrayado y azul.
Un enlace visitado está subrayado y morado.
Un enlace activo está subrayado y rojo.

Listas

Las listas se pueden crear con dos etiquetas principales:

La etiqueta <ul>, que sirve para crear listas sin ordenar y la etiqueta <ol> que sirve para crear listas ordenadas.
Además cada objeto que queramos añadir a la lista tendra que ir en la etiqueta <li>

Tablas

Una tabla HTML se define con la etiqueta <table> .
Cada fila de la tabla se define con la etiqueta <tr>. Un encabezado de tabla se define con la etiqueta <th>.
Por defecto, los encabezados de las tablas están en negrita y centrados. Una tabla de datos / celda se define con la etiqueta <td>.

Imagenes

En HTML, las imágenes se definen con la etiqueta <img>.
La etiqueta <img> está vacía, contiene solo atributos y no tiene una etiqueta de cierre.

El atributo src especifica la URL (dirección web) de la imagen. El atributo alt proporciona un texto alternativo para una imagen, si el usuario por alguna razón no puede verla (debido a una conexión lenta, un error en el atributo src o si el usuario usa un lector de pantalla).
El valor del atributo alt debe describir la imagen.

Formularios

El elemento <input> es el elemento de formulario más importante. El elemento <input> se muestra de varias maneras, según el atributo de tipo.

Observe el uso delelemento <label> en el ejemplo anterior.

La etiqueta <label> define una etiqueta para muchos elementos de formulario.
El elemento <label> es útil para los usuarios de lectores de pantalla, ya que el lector de pantalla leerá y cargará la etiqueta cuando el usuario esté enfocado en el elemento de entrada.
El elemento <label> también ayuda a los usuarios que tienen dificultades para hacer clic en regiones muy pequeñas (como botones de radio o casillas de verificación), porque cuando el usuario hace clic en el texto dentro del elemento <label>, alterna el botón de radio / casilla de verificación.
El atributo for de la etiqueta <label> debe ser igual al atributo id del elemento <input> para unirlos.

<input type="radio"> define un botón de radio.
Los botones de opción le permiten al usuario seleccionar una de una cantidad limitada de opciones.

<input type="submit"> define un botón para enviar los datos del formulario a un controlador de formulario.
El controlador de formulario suele ser una página en el servidor con un script para procesar datos de entrada.
El controlador de formulario se especifica en el atributo de acción del formulario.

Multimedias

El elemento <video> HTML5 especifica una forma estándar de incrustar un video en una página web.
El controls atributo agrega controles de video, como reproducción, pausa y volumen.

El elemento <source> le permite especificar archivos de video alternativos que el navegador puede elegir. El navegador usará el primer formato reconocido.
El texto entre las etiquetas <video> y </video> sólo se mostrará en los navegadores que no soportan el elemento <video>.

El elemento <audio> HTML5 especifica una forma estándar de incrustar audio en una página web.
El controlsatributo agrega controles de audio, como reproducción, pausa y volumen.

El elemento <source> le permite especificar archivos de audio alternativos que el navegador puede elegir. El navegador usará el primer formato reconocido.
El texto entre las etiquetas <audio> y </audio> sólo se mostrará en los navegadores que no soportan el elemento <audio>.
